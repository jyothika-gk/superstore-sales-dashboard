# Sample Superstore Sales Dashboard (Excel)

## Project Overview

This project analyzes the Sample Superstore dataset using Microsoft Excel. The goal was to explore sales performance, profit trends, and customer contributions by creating an interactive dashboard.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Data Visualization

## Dashboard Features

The dashboard provides a visual overview of sales performance with the following metrics:

* Total Sales: 2.30M
* Total Profit: 286K
* Profit Margin: 12.43%
* Total Orders: 5009

## Analysis Included

The dashboard analyzes multiple business aspects:

* Sales by Category (Furniture, Office Supplies, Technology)
* Region-wise Sales Distribution
* Top 5 Products by Sales
* Top 5 Customers
* Month-wise Sales Trends

## Key Insights

* The **Technology category generated the highest sales** among all categories.
* The **West region contributes the largest share of sales**.
* Certain products and customers contribute significantly to total revenue.
* Sales performance varies across different months.
